# Contrato de saída

```json
{
  "agent": "gram",
  "category": "inconsistency",
  "message": "Explicação curta do problema local.",
  "paragraph_index": 10,
  "issue_excerpt": "fragmento com problema",
  "suggested_fix": "fragmento corrigido"
}
```

O índice é global; o fragmento deve ser mínimo; a correção deve ser concreta. `format_spec` orienta tipografia segura.

O diagnóstico registra decisões, origem, métricas e falhas. Não medir qualidade apenas pela quantidade.
